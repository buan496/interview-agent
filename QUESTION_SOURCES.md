# Question Sources

The curated question bank contains translated or adapted prompts from the
following public interview-preparation projects. Each imported question keeps
its source URL in `source_note`.

## Front-end Developer Interview Questions

- Source: https://github.com/h5bp/Front-end-Developer-Interview-Questions
- License: MIT
- Usage: selected HTML, CSS, JavaScript, browser, network, testing, and
  performance interview prompts.

## Tech Interview Handbook

- Source: https://github.com/yangshun/tech-interview-handbook
- License: MIT
- Usage: selected Blind 75 / Grind 75 coding prompts and common behavioral
  interview prompts.

## System Design Primer

- Source: https://github.com/donnemartin/system-design-primer
- License: Creative Commons Attribution 4.0 International (CC BY 4.0)
- Usage: selected common system-design interview prompts.

## Adaptation Notice

Question wording has been translated or shortened for conversational interview
practice. Evaluation rubrics and answer keys in this repository are original
summaries written for this application. The data does not claim that a question
was asked by a specific employer unless the source explicitly documents that
claim.
